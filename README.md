## ğŸ’¡ Customization

### Updating Content

- **Personal Info**: Edit the hero section in `index.html`
- **Research/Teaching**: Update respective sections in `index.html`
- **Skills**: Modify skill categories and percentages in the skills section
- **Colors**: Change CSS variables in `style.css` (`:root` section)

### Theme Colors

Located in `style.css` under `:root`:
```css
--primary-color: #2563eb;
--secondary-color: #0ea5e9;
```

### Adding New Sections

1. Add HTML section in `index.html`
2. Add navigation link in the navbar
3. Style in `style.css`
4. Add scroll reveal if desired in `script.js`
